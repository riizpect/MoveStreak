# MoveStreak 2.0

## Highlights
- New UX-first visual polish for key daily flows.
- Clear "today status" card on home screen.
- Improved hold-to-log behavior with explicit "done today" state.
- Settings are now easier to scan with section headers.
- New history insights (7-day rate, 30-day rate, trend).
- New milestone celebrations at 7, 30, 100, and 365 days.
- Enhanced share message with milestone callout.

## Under the hood
- Extracted date helpers to `utils/date.js`.
- Extracted JSON storage helpers to `services/storage.js`.
- Added reusable `components/SettingsSection.js`.
- Added new localization copy for Swedish and English.
